## Manual Audit Checklist

A person completing this audit should capture the following from https://trinitychange.org:

### Navigation Structure
- [ ] Main navigation items (top menu)
- [ ] Secondary navigation (if any)
- [ ] Footer navigation links
- [ ] Dropdown/mega menu items

### Page Inventory
- [ ] List all pages accessible from navigation
- [ ] List all pages accessible from footer
- [ ] List any sub-pages or deep-linked content

### Homepage Sections
- [ ] Hero headline and subheadline
- [ ] Hero CTA button text and destination
- [ ] Mission statement copy
- [ ] Services listed/highlighted
- [ ] Trust signals (certifications, logos, years in business)
- [ ] Testimonial content (text only — do not copy photos or names without permission)
- [ ] Call-to-action sections
- [ ] Footer content

### Mission & About Content
- [ ] Official mission statement
- [ ] Vision statement
- [ ] Core values listed
- [ ] Agency history/founding story
- [ ] Leadership team names and titles (public-facing only)
- [ ] Community commitment language

### Services Content
- [ ] List of all services offered
- [ ] Service descriptions
- [ ] Eligibility language
- [ ] How-to-access instructions

### Regional Center & DDS References
- [ ] Regional centers named as partners
- [ ] DDS vendor number (if publicly displayed)
- [ ] Licensing information (if publicly displayed)
- [ ] IPP/ISP language used

### Contact Information
- [ ] Phone number(s)
- [ ] Email address(es)
- [ ] Physical address
- [ ] Service area (counties)
- [ ] Business hours

### Staff Information
- [ ] Team member names (public-facing)
- [ ] Team member titles
- [ ] Team member bios
- [ ] Staff photo style (professional headshots, candid, illustrated)

### Testimonials
- [ ] Client testimonial count
- [ ] Format (text, video, star rating)
- [ ] Attribution style (first name only, initials, full name, anonymous)

### Careers Content
- [ ] Open positions listed
- [ ] Pay rates displayed
- [ ] Benefits listed
- [ ] Application method (form, email, external ATS)
- [ ] Culture/values content on careers page

### Resources & Downloads
- [ ] Downloadable documents listed
- [ ] Resource guides available
- [ ] Forms available for download

### Legal Pages
- [ ] Privacy Policy URL and key sections
- [ ] Terms of Use URL and key sections
- [ ] Accessibility Statement (if present)
- [ ] Cookie consent/notice

### Technical Notes
- [ ] CMS platform (WordPress, Squarespace, etc.)
- [ ] Estimated page speed (run through PageSpeed Insights)
- [ ] Mobile responsiveness observations
- [ ] Color palette used
- [ ] Font choices
- [ ] Key UI patterns (card layouts, icon sets, CTA styles)
