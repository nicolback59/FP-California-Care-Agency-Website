## Part 2 — Local SEO Strategy

### Google Business Profile (Must Have at Launch)

- [ ] Claim and fully verify GBP
- [ ] Primary category: "Home Health Care Service"
- [ ] Secondary category: "Social Services Organization"
- [ ] Add all services in GBP services section
- [ ] Upload 10+ real photos
- [ ] Set all service area counties
- [ ] Add business hours
- [ ] Actively request Google Reviews

### Local Citation Building (NAP Consistency)

Identical Name, Address, Phone across:
- Google Business Profile, Yelp, Facebook, LinkedIn, Bing Places, Apple Maps
- Indeed and Glassdoor (for recruiting)
- DHCS / DDS vendor directories

### Service Area Pages

Create one page per county served:
```
/services/supported-living-services-sacramento-county/
/services/supported-living-services-alameda-county/
/careers/dsp-jobs-sacramento/
/careers/caregiver-jobs-fresno/
```

Each page must reference the local regional center by name.

## Part 5 — Technical SEO Checklist for Developer

| Item | Priority | Notes |
|------|----------|-------|
| SSL certificate (HTTPS) | Must Have | SiteGround includes |
| XML sitemap | Must Have | Rank Math auto-generates |
| Robots.txt | Must Have | Block portal/login from indexing |
| Page speed < 3 sec mobile | Must Have | SiteGround caching |
| Core Web Vitals passing | Must Have | Google PageSpeed Insights |
| Mobile-first design | Must Have | Elementor responsive |
| Schema markup | Nice to Have | LocalBusiness + JobPosting |
| Canonical tags | Must Have | Avoid duplicate content |
| 301 redirects | Must Have | If reusing any trinitychange.org content |
| Alt text on all images | Must Have | Accessibility + SEO |
| One H1 per page | Must Have | Heading hierarchy |
| Internal linking | Must Have | Every page linked from at least one other |
| Clean URL slugs | Must Have | No date-based or ID-based URLs |

**Recommended SEO Plugin:** Rank Math (free) — more features than Yoast free tier.
